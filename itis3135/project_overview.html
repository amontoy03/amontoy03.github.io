Client Project Overview

Project Name: A Beginner's Guide to Web Development

Project Overview:

  • The web application will serve as a guide for students who are just beginning to learn JavaScript, CSS, and HTML. Its purpose is to provide basic lessons, code examples, and challenges to help those that are interested in Web Development.
  • The intended users will be for those who are interested and beginning to learn the basics of Web Development.
  • The content of the website will contain a home page, Learn HTML page, Learn CSS page, Learn JavaScript page, and a Quiz page
