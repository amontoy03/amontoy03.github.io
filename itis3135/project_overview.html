Client Project Overview

Project Name: A Beginner's Guide to Web Development

Project Overview:

  • The web application will serve as a guide for students who are just beginning to learn JavaScript, CSS, and HTML. Its purpose is to provide basic lessons, code examples, and challenges to help those that are interested in Web Development.
  • The intended users will be for those who are interested and beginning to learn the basics of Web Development.
  • The content of the website will contain a home page, Learn HTML page, Learn CSS page, Learn JavaScript page, and a Knowledge Check page

Dynamic Functionality on the Website

The website will use JavaScript to enhance interactivity across several pages. These features aim to provide an engaging and hands-on learning experience for beginners. Each page will include a basic live code editor where users can type their code and click a Run button to see the results immediately. An example of this interactivity would be W3Schools where you can try and code in a editor.

The website will use a multiple-choice quiz will allow users to answer questions and receive immediate feedback along with their final score after submitting the quiz. Addiontionlly, JavaScript will check if all quiz questions are answered before submission and display an error if not. An example of this interactivity would be Quizizz
