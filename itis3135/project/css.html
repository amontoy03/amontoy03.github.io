The Basics To CSS:

CSS (Cascading Style Sheets) is what you use to make your website look great! You can change colors, fonts, layout, and more. Below here are a few concepts that will help you better understand:

CSS Selectors

Here are some selector types with examples:

/* Target all <p> tags */
    p {
      color: blue;
      font-size: 16px;
    }
    
    /* Style elements with the class "highlight" */
    .highlight {
      background-color: yellow;
    }
    
    /* Apply style to an element with ID "main-header" */
    #main-header {
      text-align: center;
    }
    
    /* Combine selectors or target multiple types */
    h1, h2, .intro {
      font-weight: bold;
    }
    
    /* Select nested items */
    .container .item {
      margin: 10px;
    }
    
    /* Apply to every element on the page */
    * {
      margin: 0;
      padding: 0;
    }

Common CSS Properties ▼

These are some of the most frequently used CSS properties:

  • color: Sets the text color
  • background-color: Sets the background color
  • font-size: Sets the size of the font
  • font-family: Sets the typeface of the text
  • margin: Sets the space outside the element
  • padding: Sets the space inside the element
  • border: Sets the border style, width, and color
  • width / height: Sets the dimensions of an element
  • display: Defines how an element is displayed (e.g., block, inline, flex)
