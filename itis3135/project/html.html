Learn HTML

The Basics To HTML:

HTML (HyperText Markup Language) is a language used to build the basic structure of web pages. Below here are a few concepts that will help you better understand:

This is how you start an HTML document:

            <!DOCTYPE html>
            <html>
              <head>
                <title>My First Page</title>
              </head>
              <body>
                <h1>Hello, World!</h1>
                <p>Welcome to HTML basics.</p>
              </body>
            </html>

Elements

HTML is made up of elements, which are the building blocks of a web page. An element usually includes an opening tag, some content, and a closing tag.

For example:

<p>This is a paragraph.</p>

Headings

Headings in HTML help organize content. They range from <h1> (most important) to <h6> (least important): 

          <h1>Main Heading</h1>
          <h2>Subheading</h2>
          <h3>Smaller Subheading</h3>

Tags ▼

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Anchor (Link): <a href="url">
  • Image: <img src="image.jpg" alt="Description">
  • Lists: <ul>, <ol>, <li>
  • Containers: <div> and <span>
