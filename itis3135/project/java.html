The Basics To Java:

JavaScript lets you add interactivity to your pages. You can create variables, define functions, and respond to events. Below here are a few concepts that will help you better understand:

Basic JavaScript Syntax

JavaScript can be written within <script> tags or in external .js files. Example:

let greeting = "Hi there!";
        alert(greeting);

Variables in JavaScript

Variables let you store data. You can use let, const, or var to declare them:

let username = "Alice";
        const userAge = 30;
        var enrolled = false;

Functions

Functions help organize your code into reusable blocks:

function sayHello() {
            console.log("Hello from inside a function!");
        }
        
        sayHello();
